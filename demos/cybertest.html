OSINT Test Scenarios for CTF Competition

Scenario 1: Basic Search Operators

Challenge: You need to find a PDF document about "cybersecurity training" that's hosted specifically on the University of Alabama website.

Question: What search query would you use to find this document?

Correct Answer: site:ua.edu filetype:pdf cybersecurity training

Explanation:

  • site:ua.edu limits results to University of Alabama domains
  • filetype:pdf searches only for PDF files
  • The remaining terms are the content keywords
  • This combines two essential OSINT search operators for targeted results

Scenario 2: Social Media Investigation

Challenge: You've found a social media post mentioning "Roll Tide Festival 2024" but need to verify if this is a real event at the University of Alabama.

Question: What search strategy would help you verify this information?

Correct Answer: "Roll Tide Festival 2024" site:ua.edu OR site:alabama.edu

Explanation:

  • Quotation marks ensure exact phrase matching
  • Using official university domains increases credibility
  • The OR operator searches multiple related domains
  • If no results appear on official sites, the event may be unofficial or fake

Scenario 3: Geolocation Analysis

Challenge: You found GPS coordinates in a file: 33° 12' 35.3334", -87° 32' 46.8306"

Question: What location do these coordinates point to, and what format are they in?

Correct Answer:

  • Format: DMS (Degrees, Minutes, Seconds)
  • Location: University of Alabama campus, Tuscaloosa
  • Converted to DD: 33.209815, -87.546197

Explanation:

  • DMS format uses degrees (°), minutes ('), and seconds (")
  • Negative longitude indicates Western Hemisphere
  • These coordinates specifically point to the UA campus area
  • Converting between coordinate formats is crucial for cross-referencing locations

Scenario 4: Advanced Search Techniques

Challenge: Find information about a person named "Dr. Sarah Johnson" who works in cybersecurity, but exclude results from LinkedIn and Facebook.

Question: Construct an effective search query.

Correct Answer: "Dr. Sarah Johnson" cybersecurity -site:linkedin.com -site:facebook.com

Explanation:

  • Quotation marks ensure the exact name is searched
  • The minus (-) operator excludes specific sites
  • This helps find more professional or academic sources
  • Useful when social media results dominate search results

Scenario 5: Image Investigation

Challenge: You have a photo that appears to be taken at a university. The image shows a distinctive clock tower, but you need to determine which university.

Question: What OSINT approach would help identify the location?

Correct Answer: Multiple approaches:

  1. Reverse Image Search: Upload to Google Images, TinEye, or Yandex
  2. Architectural Search: university "clock tower" Alabama OR Auburn OR UAB
  3. Visual Analysis: Look for visible text, logos, or distinctive architectural features

Explanation:

  • Reverse image searches can find identical or similar images with location data
  • Searching for distinctive architectural features helps narrow locations
  • Combining multiple search terms with OR operators expands results
  • Always verify findings with multiple sources

Scenario 6: Metadata Investigation

Challenge: You've downloaded a document that claims to be from "Crimson Defense Cyber Security Club" but need to verify its authenticity.

Question: What OSINT techniques would help verify this document?

Correct Answer:

  1. Search for official sources: site:uacrimsondefense.github.io OR site:ua.edu "Crimson Defense"
  2. Cross-reference information: Compare document details with official club information
  3. Check document properties: Examine metadata for creation details
  4. Verify contact information: Confirm any email addresses or websites mentioned

Explanation:

  • Official sources provide ground truth for verification
  • Metadata can reveal creation software, dates, and author information
  • Cross-referencing helps identify inconsistencies
  • Always verify through multiple independent sources

Scenario 7: Username Investigation

Challenge: You found a username "CrimsonHacker2025" associated with suspicious activity. You need to find other online accounts with this username.

Question: What search strategy would help find associated accounts?

Correct Answer:

  1. Exact username search: "CrimsonHacker2025"
  2. Platform-specific searches:
    • site:twitter.com "CrimsonHacker2025"
    • site:github.com "CrimsonHacker2025"
    • site:reddit.com "CrimsonHacker2025"
  3. Username variation searches: CrimsonHacker OR "Crimson Hacker" 2025

Explanation:

  • Quotation marks prevent search engines from breaking up the username
  • Site-specific searches help find accounts on major platforms
  • Username variations account for different formatting preferences
  • Systematic searching across platforms builds a comprehensive profile

Scenario 8: Event Verification

Challenge: Someone claims there was a "UACTF 2024" competition, but you need to verify when it actually occurred and find official results.

Question: How would you verify this information?

Correct Answer:

  1. Official source search: site:crimsondefense.cs.ua.edu "UACTF 2024"
  2. News coverage: "UACTF 2024" OR "University of Alabama CTF" news
  3. Social media verification: "UACTF 2024" site:twitter.com OR site:facebook.com
  4. Academic calendar check: site:ua.edu calendar events "capture the flag"

Explanation:

  • Official university sources provide authoritative information
  • News coverage offers independent verification
  • Social media posts can provide real-time event coverage
  • Academic calendars help verify official university involvement

Quick Tips for OSINT Success:

  1. Always verify through multiple independent sources
  2. Use quotation marks for exact phrases
  3. Combine search operators for precision
  4. Check official sources first
  5. Document your search process
  6. Be aware of information freshness and relevance
  7. Respect privacy and legal boundaries

These scenarios test fundamental OSINT skills while focusing on realistic cybersecurity contexts that students might encounter in actual competitions.
