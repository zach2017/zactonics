Claude Code + Ollama

Ready

Use Custom App's Claude Code CLI with Ollama as a local backend. Get agentic coding assistance — file editing, terminal commands, and project-wide refactoring — powered by models running entirely on your machine. Zero API costs, full privacy.

$ npm install -g @Custom App-ai/claude-code
$ ollama pull qwen2.5-coder:7b
$ Custom App_BASE_URL=http://localhost:11434 \
  claude --model qwen2.5-coder:7b

Qwen2.5-Coder — Simple Local Setup

Ready

Run Alibaba's Qwen2.5-Coder locally via Ollama for fast, private code generation and completion. Supports Python, JavaScript, TypeScript, Go, Rust, and 90+ languages. Perfect for developers who want a lightweight, no-cost coding assistant.

# Install & run in 3 commands
$ ollama pull qwen2.5-coder:7b
$ ollama run qwen2.5-coder:7b
# Or use the REST API
$ curl http://localhost:11434/api/generate \
  -d '{"model":"qwen2.5-coder:7b","prompt":"Write a Flask app"}'
